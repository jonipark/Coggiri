### What it is

**Coggiri** is a Vision Pro experience built to improve **cognitive engagement for seniors** through intuitive hand-based gameplay.  
Players grab floating fruits in 3D space to train **attention, reaction speed, and motor coordination**, blending play with wellness.

---

### My role (Swift Developer)

- **Gameplay Design:** Built fruit grab and scoring mechanics using RealityKit.  
- **Audio System:** Implemented adaptive SFX and background music for engagement.  
- **Data Integration:** Synced scores with Notion database - https://coggiri.xyz

---

### Key Features
- Hand gesture-based grabbing using RealityKit collision detection.  
- Dynamic round system with countdown bar + visual urgency cues.  
- Real-time score feedback and result upload to Notion leaderboard.  
- Simple, readable Korean text interface for senior-friendly play.  
- Adaptive audio feedback ‚Äî motivational chime, combo sound effects.
